Detailed Description

An implementation of HTTPBodyStream that combines an array of several other HTTPBodyStream objects into a single, unified stream.

explicit

Creates a stream from an array of other stream parts.

Parameters
[in]partsA vector of HTTPBodyStream objects, of which this object takes ownership, that will be represented as a single unified stream. Callers should not mutate the stream objects after passing them to an instance of this class.
